# Phoca PDF



Phoca PDF is Joomla! CMS component. It helps to set and create PDF documents in Joomla!. The whole Phoca PDF framework contains: 
- Phoca PDF component 
- Phoca PDF fonts 
- Phoca PDF plugins.
